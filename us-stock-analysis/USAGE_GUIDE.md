## 🎯 10种专业分析模式

### 1️⃣ 完整股票分析
**使用场景**: 需要全面了解一家公司

```
完整股票分析 [股票代码]
```

**输出包含:**
- 市场概况（股价、涨跌幅、市值）
- 基本面分析（收入、利润、增长率）
- 估值分析（P/E、P/B、EV/EBITDA）
- 业务质量（商业模式、竞争优势）
- 财务健康（债务、现金流、流动性）
- 风险因素
- 投资建议（评级、目标价）

**示例:**
```
完整股票分析 NVDA
```

---

### 2️⃣ 财务报表深度分析
**使用场景**: 深入理解公司财务状况

```
财务报表深度分析 [公司名称/代码]
```

**输出包含:**
- 损益表分析（收入趋势、成本结构）
- 资产负债表分析（资产/负债结构、关键比率）
- 现金流量表分析（经营/投资/筹资现金流）
- 盈利质量分析（GAAP vs 非GAAP）
- 趋势和异常识别

**示例:**
```
财务报表深度分析 Amazon (AMZN)
```

---

### 3️⃣ 财报分析器
**使用场景**: 分析最新财报发布后的影响

```
财报分析器 [公司名称/代码]
```

**输出包含:**
- 核心数据表现（收入、EPS vs 预期）
- 前瞻指引（管理层指引 vs 市场预期）
- 业务亮点（增长驱动、新产品）
- 管理层评论（CEO/CFO讲话）
- 市场反应（股价、分析师变化）
- 关键关注点

**示例:**
```
财报分析器 Microsoft 最新财报
```

---

### 4️⃣ 行业与板块比较
**使用场景**: 对比多家公司，选择最佳投资标的

```
行业比较 [股票1] vs [股票2] vs [股票3]
```

**输出包含:**
- 基础对比表（股价、市值、涨幅）
- 财务表现对比（收入、利润率、ROE）
- 估值对比（P/E、P/B、EV/EBITDA）
- 财务健康对比（债务、流动性）
- 现金流对比
- 相对优势分析
- 排名和推荐

**示例:**
```
行业比较 TSLA vs F vs GM
```

---

### 5️⃣ 估值模型构建器
**使用场景**: 构建专业的估值模型

```
估值模型 [公司名称/代码]
```

**输出包含:**
- DCF 分析（现金流折现法）
  * 收入假设、利润率假设
  * WACC 计算
  * 自由现金流预测
- 可比公司分析（相对估值法）
- 历史估值分析
- 三情景分析（牛市/基准/熊市）
- 敏感性分析
- 关键风险

**示例:**
```
估值模型 Apple
```

---

### 6️⃣ 股息和被动收入分析器
**使用场景**: 分析股息投资和被动收入策略

```
股息分析 [公司名称/代码]
```

**输出包含:**
- 当前股息档案（收益率、支付频率）
- 股息增长历史（增长率、连续年数）
- 可持续性检查（派息比率、现金流覆盖）
- 同行比较
- 收入预测（5/10/20年）
- 风险评估（削减风险因素）

**示例:**
```
股息分析 Johnson & Johnson (JNJ)
```

---

### 7️⃣ 风险与红旗扫描器
**使用场景**: 识别投资风险和危险信号

```
风险分析 [公司名称/代码]
```

**输出包含:**
- 财务健康风险（债务、现金流、利润率）
- 内部人和机构活动（交易、离职、持仓变化）
- 业务集中风险（客户、产品、地理）
- 竞争威胁（主要竞争对手）
- 监管和法律（诉讼、调查、罚款）
- 会计质量检查（GAAP差异、重述）
- 宏观敏感性（利率、经济周期、汇率）
- 风险评级和推荐

**示例:**
```
风险分析 Tesla
```

### 8️⃣ ETF 和投资组合分析器
**使用场景**: 分析 ETF 或投资组合

```
组合分析 [ETF代码或持股列表]
```

**输出包含:**
- 配置分析（板块、地理、市值、风格）
- 持股分析（前10大持仓、集中度）
- 风险指标（Beta、波动率、夏普比率）
- 成本分析（费用比、年度成本）
- 收入分析（股息收益率）
- 压力测试（历史危机时期表现）
- 优化建议

**示例:**
```
组合分析 VOO
```

或

```
组合分析 AAPL 30%, MSFT 20%, GOOGL 15%, AMZN 15%, TSLA 20%
```

### 9️⃣ 宏观和市场情绪扫描器
**使用场景**: 了解宏观环境对投资的影响

```
宏观分析 [板块/股票/投资组合]
```

**输出包含:**
- 美联储和利率（联邦基金利率、未来会议、CME概率）
- 通胀（CPI、PCE数据、趋势）
- 经济健康（GDP、失业率、消费者信心、PMI）
- 市场内部（指数表现、市场广度、VIX、情绪指标）
- 板块轮动（流入流出板块）
- 地缘政治和事件日历（未来风险、关键事件）
- 战略判决（市场环境、投资策略）

**示例:**
```
宏观分析 科技股
```

## ❓ 常见问题

**Q: 为什么某些数据显示"未公开"?**
A: 某些公司或指标可能不公开特定数据，或数据源暂时不可用。

**Q: 如何理解估值的三情景分析?**
A:
- 牛市场景: 乐观假设下的目标价（概率30%）
- 基准场景: 基于最可能情况的目标价（概率50%）
- 熊市场景: 悲观假设下的目标价（概率20%）
- 加权目标价: 综合三场景的概率加权平均

**Q: 分析报告中的评级标准是什么?**
A:
- 买入: 预期12个月上涨潜力 >15%
- 持有: 预期12个月变动在 ±15% 范围
- 卖出: 预期12个月下跌风险 >15%

**Q: 如何解读风险扫描器的红旗标记?**
A: 红旗标记表示潜在的重大问题，需要深入调查。多个红旗标记可能表明公司存在严重问题。

**Q: 分析报告多久更新一次?**
A: 建议在财报发布、重大事件或市场重大变化后更新分析。
