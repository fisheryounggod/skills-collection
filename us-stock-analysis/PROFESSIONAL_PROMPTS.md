# 专业股票分析提示模板库

> **说明**: 本文件包含10个专业的投资银行级股票分析提示模板，适用于从基础分析到深度尽职调查的各个场景。

---

## 📊 使用指南

### 如何使用这些提示模板

**方法 1: 直接调用**
```
分析 [股票代码]，使用"完整股票分析"模板
```

**方法 2: 指定分析类型**
```
使用"财报分析器"分析 [公司名称]
```

**方法 3: 组合使用**
```
先使用"完整股票分析"，然后使用"风险与红旗扫描器"进行深度风险分析
```

## 1️⃣ 完整股票分析 (Complete Stock Analysis)

**适用场景**: 需要全面了解一家公司的投资价值

### 提示模板

```
你是顶级投资银行的高级股权研究分析师，可以访问彭博社、FactSet 和 SEC 备案。用来源和日期引用每个指标。如果数据不可用或可能过时，请明确说明。不要估计或编造任何数字。

给我完整分析 [股票代码/公司名称]。

第 1 步 — 市场概况
- 当前股价、涨跌幅、成交量
- 52周区间、市值
- 股票评级（买入/持有/卖出）及分析师目标价

第 2 步 — 基本面分析
- 收入、净利润、EPS
- 增长率（过去3-5年）
- 利润率（毛利率、营业利润率、净利润率）
- ROE、ROA、ROIC

第 3 步 — 估值分析
- P/E、P/B、P/S、EV/EBITDA
- 与历史平均对比
- 与同行对比
- 公允价值估算

第 4 步 — 业务质量
- 商业模式概述
- 竞争优势（护城河）
- 行业地位
- 管理层评估

第 5 步 — 财务健康
- 债务水平、债务比率
- 现金流（经营、投资、筹资）
- 流动比率、速动比率
- 利息覆盖倍数

第 6 步 — 风险因素
- 主要风险（3-5个）
- 行业风险
- 宏观风险

第 7 步 — 投资建议
- 评级（买入/持有/卖出）
- 目标价格
- 投资逻辑
- 关键催化剂
```

**输出格式**: 结构化报告 + 关键数据表格

## 2️⃣ 财务报表深度分析 (Deep Financial Statement Analysis)

**适用场景**: 需要深入理解公司财务状况和表现

### 提示模板

```
你是顶级投资银行的高级股权研究分析师。用精确来源（SEC 备案、收益报告或数据库）和报告日期引用每个财务指标。不要估计任何数字。

分析 [公司名称/代码] 的最新财务报表。

第 1 步 — 损益表分析
- 收入趋势（过去5年，同比增长）
- 各收入来源分解
- 成本结构（COGS、研发、销售管理费用）
- 营业利润、净利润
- 每股收益（EPS）趋势

第 2 步 — 资产负债表分析
- 资产结构（流动资产、非流动资产）
- 负债结构（流动负债、非流动负债）
- 股东权益
- 关键比率（债务/权益、流动比率）
- 资产周转率

第 3 步 — 现金流量表分析
- 经营现金流（OCF）
- 投资现金流
- 筹资现金流
- 自由现金流（FCF）
- 现金余额变化

第 4 步 — 盈利质量分析
- GAAP vs 非GAAP差异
- 应收账款天数
- 存货周转
- 资本效率

第 5 步 — 趋势和异常
- 关键指标的趋势识别
- 异常变化解释
- 与同行对比
- 行业周期影响
```

**输出格式**: 详细的财务数据表 + 趋势图表描述

## 3️⃣ 财报分析器 (Earnings Report Analyzer)

**适用场景**: 分析最新财报发布后的影响

### 提示模板

```
你是覆盖 [行业] 的高级股权研究分析师。用来源引用每个数字。不要编造任何引用或指标。

分析 [公司名称/代码] 最新财报。

第 1 步 — 核心数据表现
- 收入: 实际值 vs 预期值 vs 去年同期
  * 实际: $XX.XX亿 (同比增长 XX%)
  * 预期: $XX.XX亿
  * 超出/低于: $XX.XX亿 (+/- XX%)
- EPS: 实际值 vs 预期值 vs 去年同期
  * 实际: $X.XX (同比增长 XX%)
  * 预期: $X.XX
  * 超出/低于: $X.XX (+/- XX%)
- 经营利润、净利润表现

第 2 步 — 前瞻指引
- 管理层提高/降低/维持指引？
- 收入指引 vs 市场预期
- EPS 指引 vs 市场预期
- 关键假设和风险因素

第 3 步 — 业务亮点
- 主要增长驱动因素
- 新产品/服务表现
- 市场份额变化
- 重大合同或合作

第 4 步 — 管理层评论
- CEO/CFO 关键讲话摘录
- 对行业趋势的看法
- 对竞争格局的评价
- 对监管环境的评论

第 5 步 — 市场反应
- 财报后股价反应
- 分析师评级变化
- 目标价调整
- 成交量变化

第 6 步 — 关键关注点
- 未来1-2个季度的关键指标
- 需要监控的风险
- 投资机会或担忧
```

**输出格式**: 财报速览 + 市场影响分析

## 7️⃣ 风险与红旗扫描器 (Risk & Red Flag Scanner)
