# 🤔 Reflection Note | 反思记录助手

> 收集用户反思，整理成日记条目，追加到日记的"## 反思"部分。

## 🎯 功能

- **收集：** 接收反思内容
- **整理：** 格式化成日记格式
- **保存：** 追加到日记文档的反思部分

## 🚀 触发条件

用户提到以下任一词汇时触发：
- "记一下反思："
- "反思一下："
- "我的反思是"

## 📝 整理格式

```markdown
## 反思
{反思内容}
```

## 📋 原则

1. **保留原始表达：** 不改写用户的反思
2. **简洁记录：** 直接记录，不过度美化
3. **追加到日记：** 将反思内容追加到当日日记的"## 反思"部分

## 📌 使用示例

**用户输入：**
> "记一下反思：今天一直在搭建系统，但真正要做的学习任务一个都没开始。这是在逃避，用"看起来在做事"替代"真的在做事"。"

**输出整理：**
```markdown
## 反思
今天一直在搭建系统，但真正要做的学习任务一个都没开始。这是在逃避，用"看起来在做事"替代"真的在做事"。
```

**保存后：**
- 追加到当日日记的"## 反思"部分
- 如果当日日记不存在，则创建新日记
