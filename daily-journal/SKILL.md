# 📅 Daily Journal | 日记助手

> 收集用户日记，整理成格式化笔记，保存到飞书知识库。

## 🎯 功能

- **收集：** 接收用户输入，不中断
- **确认：** 完成时询问"还有要记的吗？"（只问一次）
- **整理：** 按固定格式组织内容
- **保存：** 自动保存到飞书知识库

## 🚀 触发条件

用户提到以下任一词汇时触发：
- "日记"
- "复盘"
- "今天做了什么"
- "/daily"
- "/date"
- "/day"

## 📝 整理格式

```markdown
# 📅 {日期} 日记

## 今日事项
- 简洁条目列出

## 状态 & 感受
- 1-2 句（用户有提到时）

## 反思
- 用户的思考（有提到时）
```

## 📋 原则

1. **保留原始表达：** 不过度美化
2. **300字以内：** 保持简洁
3. **不添加内容：** 不写入用户没说的东西
4. **只问一次：** 完成时只问"还有要记的吗？"
5. **文件存在检查：** 保存前检查是否有同名文档，存在则追加内容

## 🔄 工作流程

1. **触发** → 检测到日记关键词
2. **收集** → 接收用户输入
3. **整理** → 按格式化内容组织
4. **检查文件** → 查找同标题文档
5. **保存** → 创建或追加到飞书文档
   - 文件不存在 → 创建新文档
   - 文件存在 → 追加到已有文档

### 文件存在性检查

**检查逻辑：**
- 根据当前日期搜索文档标题：`{日期} 日记`
- 如果找到同名文档 → 使用 `append` 模式
- 如果未找到 → 使用 `create` 模式

**追加格式：**
```markdown
---
{时间戳} 新增内容：
{整理后的日记内容}
```
