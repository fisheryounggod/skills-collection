# ABC-B2C 经济学研究范式 · 文献分析技能

## 核心理论框架

### 一、ABC 分析范式（底层要素分解）

```
任何经济学问题都可以分解为三个底层要素：

  A (Agent)      — 市场主体
                  谁在做决策？消费者、企业、政府、工人、投资者……
                  → 识别决策单元

  B (Behavior)   — 主体的行为
                  主体做了什么？消费、储蓄、投资、生产、出口、结婚、犯罪……
                  → 识别被解释变量/结果变量

  C (Constraint) — 主体面临的约束
                  什么约束了主体的选择？价格、收入、制度、信息、时间、注意力……
                  → 识别解释变量/因果变量

三者共同构成经济学问题的底层要素。
```

**标注格式：**
```
@A(消费者) @B(增加储蓄) @C(利率上升)
@A(企业)   @B(减少出口) @C(关税壁垒)
@A(工人)   @B(劳动力流动) @C(户籍制度)
```

### 二、C2B 范式（科学问题提出）

```
C2B 读作 "C to B" — 核心因果问题格式：

    "C 对 B 有什么影响？"

    C 是 B 的因，B 是 C 的果
    这是计量经济学中待检验假说的基本形式

一旦分解出 ABC 三要素，就很容易用 C2B 范式提出相应的科学问题。
```

**问题变体：**
```
基准格式：  C 对 B 有什么影响？
反事实：    如果没有 C，B 会怎样？
机制分析：  C 通过什么渠道影响 B？
异质性：    C 对不同 A 的 B 影响是否不同？
动态分析：  C 对 B 的影响随时间如何演变？
```

### 三、文献分析四步法

```
步骤 1 · ABC 提取
  → 从摘要/引言中提取核心 ABC 三要素

步骤 2 · C2B 问题重构
  → 将研究问题转化为"C对B有什么影响"的标准形式

步骤 3 · 识别策略评估
  → 评估作者使用了什么识别策略（DID、IV、RDD、匹配的、实验的……）
  → 该策略是否干净排除了其他解释？

步骤 4 · ABC-C2B 延伸
  → 提出 3-5 个 C2B 形式的延伸研究问题
```

## 应用场景示例

### 场景 1：关税对企业出口的影响

```
@A(出口企业)
@B(出口额 / 出口决策)
@C(关税税率)

C2B问题：关税税率上升对出口企业的出口额有什么影响？

识别策略：利用中美贸易战的准自然实验（双重差分）
反事实假设：未受贸易战影响的企业作为对照组
核心威胁：企业自选择问题（哪些企业被加关税？）

C2B延伸：
1. 关税对不同规模企业的出口影响是否存在异质性？
2. 关税通过什么渠道影响出口？（价格效应 vs 数量效应）
3. 关税对出口企业就业有什么影响？（间接效应）
```

### 场景 2：最低工资对就业的影响

```
@A(低技能工人 / 餐饮业企业)
@B(就业水平 / 雇用决策)
@C(最低工资标准)

C2B问题：最低工资提高对低技能工人就业有什么影响？

识别策略：利用各州最低工资差异（断点回归或双重差分）
核心假设：最低工资外生于当地劳动力市场条件

C2B延伸：
1. 最低工资对不同年龄段工人的就业影响是否存在差异？
2. 最低工资通过什么渠道影响就业？（劳动力需求 vs 劳动力供给）
3. 最低工资对工资分布的溢出效应（spillover）有多大？
```

## 输出规范

1. **结构化输出**：严格按上述模板格式输出
2. **中文优先**：所有分析输出使用中文，术语保留英文缩写
3. **证据分级**：区分核心发现（主要结果表）与辅助发现（附录/稳健性）
4. **不替代理性**：分析揭示可能性，最终判断留给人
